# Modal Task View
**Version:** 1.0.0

Plugin that allows to open the task view in a modal window instead of a new page. It also adds navigation buttons to switch between modal and full view.

## Author
- laimeilleurzamis
- License MIT

## Features
This plugin improves the user experience by keeping you on the board when consulting tasks:
- **Modal View:** Clicking on a task from the board opens its details in a quick modal window instead of redirecting to a new page.
- **Custom Navigation:** Replaces the default modal close button with a custom top navigation bar.
- **View Switching:** Includes dedicated buttons to easily switch the task view between modal mode and fullscreen window mode.
- **Quick Edit:** Features a shortcut to edit the task directly from the new navigation bar.

## Requirements
- Kanboard >= 1.0.35

## Installation
You have the choice between 3 methods:

1. Install the plugin from the Kanboard plugin manager in one click.
2. Download the zip file and decompress everything under the directory `plugins/ModalTaskView`.
3. Clone this repository into the folder `plugins/ModalTaskView`.

> **Note:** The plugin folder is case-sensitive.
